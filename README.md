# Chuks Kitchen - Authentic Nigerian Cuisine 🍲

**Chuks Kitchen** is a modern, responsive web application designed for a premium Nigerian restaurant. It allows users to explore various food categories—from Jollof Rice to Swallows and Grills—and manage their orders with a sleek, user-friendly interface.

---

## 🚀 Features

* **Dynamic Menu Explorer:** Organized food categories (Popular, Jollof, Swallows, Grills, etc.).
* **Responsive Product Cards:** Each dish includes high-quality imagery, descriptions, pricing, and a quick "Add to Cart" button.
* **Sidebar Navigation:** An easy-to-use menu category sidebar that highlights the active section.
* **Hero Landing Section:** A visually striking intro section with a darkened overlay for high readability.
* **Fully Responsive:** Designed to adapt from desktop to mobile screens using modern CSS Flexbox.

---

## 🛠️ Built With

* **HTML5:** Semantic markup for structure and accessibility.
* **CSS3:** Custom styling using HSL color variables and Flexbox for layout management.
* **Git/GitHub:** Version control and remote repository hosting (`origin`).

## ⚙️ Installation & Usage

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
```bash
git clone https://github.com/General4904/chuks-kitchen.git

```


2. **Navigate to the project folder:**
```bash
cd chuks-kitchen

```


3. **Open the project:**
Double-click `index.html` or use a "Live Server" extension in VS Code to view the site in your browser.
